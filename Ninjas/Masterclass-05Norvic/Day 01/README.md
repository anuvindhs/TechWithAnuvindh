## Codes

### For Git

- Generate Key: 
    > ssh-keygen -t ed25519 -C "your_email@example.com 

- Check keys:
    > ls -al ~/.ssh

- Copy Key: 
    > cat ~/.ssh/id_ed25519.pub

- Authenticate the Connectivity:
    > ssh -T git@github.com

-   Clone Repository
    > git clone https://github.com/anuvindhs/TechWithAnuvindh.git
